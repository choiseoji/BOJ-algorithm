## 이분탐색

[풀이]
- 처음엔 입력받은 높이 중에서 가장 높은 높이부터 -1을 해보면서 다 계산해보려고 했는데
- 이렇게 하는 것보다 이분탐색을 사용하여 적정 위치를 찾는것이 더 효율적이라고 생각했다!!
